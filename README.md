# customer_behavior_analysis
Data analytics project showcasing customer behavior analysis using python , sql and power Bi

## Overview
This project delivers a complete data analytics workflow from raw data to business insight. It combines Python, SQL, and Power BI to analyze data, identify key patterns, and present clear recommendations.

The objective is simple: support decision-making with reliable data and clear visuals

## Business Objective
The project answers key business questions such as:
- What trends drive performance over time?
- Which categories or segments contribute the most?
- Where do inefficiencies or data issues exist?

It focuses on turning data into decisions, not just analysis.

## Dataset
The dataset includes structured records relevant to business performance (e.g. sales, customers, operations).

Key characteristics:
- Mix of numerical and categorical variables  
- Missing values and inconsistencies  
- Data suitable for aggregation and trend analysis

## Tools
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- MySQL Server  
- Power BI  
- Gamma (presentation)

## Workflow
### 1. Data Loading
- Import dataset into Python  
- Validate structure, data types, and completeness  

### 2. Exploratory Data Analysis (EDA)
- Analyze distributions and relationships  
- Detect trends, patterns, and anomalies  
- Create visual summaries for key variables  

### 3. Data Cleaning
- Handle missing values with appropriate strategies  
- Remove duplicates to ensure data accuracy  
- Standardize formats for consistency  
- Prepare dataset for SQL and BI tools  

### 4. SQL Analysis
- Load cleaned data into MySQL  
- Execute queries to extract insights  
- Perform joins, aggregations, and filtering  
- Validate findings across multiple queries  

### 5. Dashboard Creation
- Build an interactive dashboard in Power BI  
- Design clear and intuitive visuals  
- Highlight KPIs and performance trends  
- Enable user-driven exploration with filters  

### 6. Reporting
- Summarize findings in a structured report  
- Present insights using Gamma  
- Provide clear, data-backed recommendations

## How to Run
1. Clone the repository  
2. Install required libraries:pip install pandas numpy matplotlib seaborn mysql-connector-python
3. Run Python scripts for data loading, EDA, and cleaning  
4. Load cleaned data into MySQL and execute SQL queries  
5. Open the Power BI file to explore the dashboard  
6. Review the report and presentation
